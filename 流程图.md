## 4. 混合检索子系统 (Hybrid Search RAG)

```mermaid
flowchart TD
    QUERY["用户查询<br/>course_code / user_message"]

    QUERY --> BM25["BM25 稀疏匹配<br/>tokenize → BM25Okapi → rank"]
    QUERY --> DENSE["Dense 语义检索<br/>text-embedding-3-small<br/>→ Milvus cosine similarity"]

    BM25 --> |"k*3 候选"| RRF
    DENSE --> |"k*3 候选"| RRF

    subgraph RRF["RRF 融合<br/>Reciprocal Rank Fusion"]
        direction TB
        SCORE["score(d) = 0.3 × 1/(60+r₁)<br/>         + 0.7 × 1/(60+r₂)"]
    end

    RRF --> THRESHOLD{"score ≥ 0.5?"}
    THRESHOLD -->|是| TOP5["Top-5 chunks<br/>→ rag_context"]
    THRESHOLD -->|否| EMPTY["rag_empty=True<br/>prompt: '不编造'"]

    BM25_FALLBACK{"BM25 可用?"}
    BM25_FALLBACK -->|No| FALLBACK["回退纯 Dense 检索"]
    FALLBACK --> THRESHOLD
```

## 5. PDF 文档分析子系统

```mermaid
sequenceDiagram
    participant User as 📱 用户
    participant Bot as bot.py

    participant Celery as Celery OCR Worker

    Bot->>Bot: 捕获 update.message.caption

    Note over Graph: analyze_document_node
    Graph->>Graph: fitz 提取全文
    Graph->>Celery: analyze_document_task.apply_async()
    Celery->>Celery: len(text) > 8000 ?
    
    alt 长文档
        Celery->>Celery: RecursiveCharacterTextSplitter<br/>(3000/200)
        Celery->>Celery: 每块 ChatGPT 摘要 (第一轮)

        Celery->>Celery: ChatGPT 直接摘要

    Note over Graph: 注入对话 pipeline
    Graph->>Graph: rag_context = 全文 + AI Summary
    Graph->>Graph: retrieve_memory → build_prompt → call_llm
    Graph->>Graph: ChatGPT 回答 "只分析第三章"
    Graph->>Graph: store_memory → conversation_memory
    
    Graph-->>Bot: final_response
    Bot-->>User: "第三章主要讲 Docker 容器化..."
```

## 7. 核心调用链

```mermaid
sequenceDiagram
    participant User as 📱 用户
    participant Bot as bot.py Handler
    participant Graph as LangGraph
    participant Milvus_C as Milvus course_documents
    participant Milvus_M as Milvus conversation_memory
    participant LLM as Azure OpenAI ChatGPT

    User->>Bot: 发送消息 / 上传文件
    Bot->>Bot: RateLimiter + Validation
    Bot->>Bot: 构建 AgentState
    Bot->>Graph: ainvoke(state)

    Note over Graph: classify_intent_node
    alt intent = retrieve_course / general_chat
        Graph->>Milvus_C: retrieve_hybrid_with_scores(query)
        Milvus_C-->>Graph: Top-5 chunks + scores
    else intent = analyze_document
        Graph->>Graph: 自注入 rag_context (PDF全文+摘要)
        Graph->>Milvus_C: async ingest_text() (后台)
    end

    Note over Graph: retrieve_conversation_memory_node
    Graph->>Milvus_M: similarity_search(query, expr=user_id)
    Milvus_M-->>Graph: 相关历史对话

    Note over Graph: build_prompt_node
    Graph->>Graph: rag_context + 记忆 + user_message
    Note over Graph: call_llm_node
    Graph->>LLM: augmented_prompt
    LLM-->>Graph: final_response

    Note over Graph: store_conversation_memory_node
    Graph->>Milvus_M: add_documents(user_msg + resp)

    Graph-->>Bot: final_response
    Bot-->>User: 回复消息
```

## 8. 限流与容错机制

```mermaid
flowchart TD
    USER["用户请求"] --> RL{"RateLimiter<br/>30次/60秒?"}
    RL -->|超过| REJECT["拒绝: '请等待'"]
    RL -->|正常| SEM{"Semaphore<br/>≤50 并发?"}
    SEM -->|超过| QUEUE["排队等待"]
    SEM -->|正常| TIMEOUT{"timeout=30s<br/>超时?"}
    TIMEOUT -->|超时| TO["返回: '请求超时'"]

    TIMEOUT -->|正常| LLM_CALL["call_llm_node"]
    LLM_CALL --> CB{"Circuit Breaker<br/>熔断器状态?"}
    CB -->|OPEN| REJECT_CB["直接返回: '服务暂时不可用'"]
    CB -->|CLOSED| LLM_API["Azure OpenAI API"]
    CB -->|HALF_OPEN| TRY["放行 1 个试探请求"]
    TRY --> LLM_API

    LLM_API -->|成功| CLOSE["熔断器 → CLOSED"]
    LLM_API -->|连续5次失败| OPEN_CB["熔断器 → OPEN (30s)"]

    LLM_API -->|失败| RETRY{"3次重试<br/>2s→4s→8s?"}
    RETRY -->|重试| LLM_API
    RETRY -->|耗尽| FAIL["返回错误消息"]
```

## 9. 视频工作流

```mermaid
sequenceDiagram
    participant User as 📱 用户
    participant Bot as bot.py
    participant Graph as LangGraph
    participant Worker as Celery Worker
    
    Note over User,Worker: 第 1 次图执行
    User->>Bot: /video
    Bot->>Graph: ainvoke
    Graph->>Graph: video_command_node
    Graph-->>Bot: set waiting_for_video_image=True
    Bot-->>User: "请发送一张图片"

    Note over User,Worker: 第 2 次图执行
    User->>Bot: [发送图片]
    Bot->>Graph: ainvoke (intent=receive_video_image)
    Graph->>Graph: 下载图片 → base64
    Graph->>Worker: analyze_image_task()
    Worker-->>Graph: AI 分析 + 3个prompt
    Graph-->>Bot: 显示分析结果 + 建议
    Bot-->>User: "请选择动画效果 1/2/3"

    Note over User,Worker: 第 3 次图执行
    User->>Bot: "1" 或 "自定义prompt"
    Bot->>Graph: ainvoke (intent=receive_video_prompt)
    Graph->>Graph: 解析 prompt 选择
    Graph->>Worker: generate_video_task()
    Graph-->>Bot: "视频生成已开始"
    Bot-->>User: "视频在后台处理中..."

    Note over User,Worker: 后台监控
    loop 每 10 秒
        Worker->>Worker: 检查 task.ready()
    end
    Worker-->>User: "视频已就绪!" + 发送视频
```
