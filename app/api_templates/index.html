🚀 Quick Start
  1. Prepare a CSV/JSON file with course or assignment data
  2. Upload it via the form on the left
  3. Data is chunked, embedded, and stored in Milvus
  4. Users can now query this data through the Telegram bot
CSV format:
course_code,course_name,class_time,location,description

💡 Batch Import from CLI

Also available via command line:

curl -X POST http://localhost:8000/api/upload \
  -F "file=@courses.csv" \
  -F "source_type=courses" curl -X POST http://localhost:8000/api/ingest \
  -H "Content-Type: application/json" \
  -d @courses.json
